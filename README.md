# CppTemplate

A minimalistic, dependency-free, and out-of-source C++ project template.

## Features

### Source Code

- CMake and CTest compatible project: `(include|src|test)/*.(hpp|cpp)` and `CMakeLists.txt`
- Markdown docs: `(CHANGELOG|LICENSE|README).md`
- `.gitignore`
